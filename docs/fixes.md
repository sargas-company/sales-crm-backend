fields — контролює відображення рядка:
- false → рядок прихований (навіть якщо value є)
- true → показується як фіксована сума
- "%" → показується як відсоток

tax, discounts, shipping — самі значення (число).

Поточний код правильний для show/hide через булеві showTax/showDiscounts/showShipping. Але є один нюанс — якщо showTax = true, API рендерить як фіксовану суму. Якщо потрібен %, треба передавати рядок "%"
замість true.
